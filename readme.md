# AIRATS

## ACSII Image Convertor

A simple Flask web app  to generate ASCII art from an image URL

The Windows app.exe is built to only run the web app in a browser. 

Double click the app.exe to launch it. 


## Installation

Open a web browser and visit [http://localhost:8081] or (http://Your-IP-Address_on_your_network:8081) to access the app.


** folder should contain all dependency's needed but you may need to install in power shell or terminal 

python

and 

see requirements.txt for more details.

    JPEG: .jpg, .jpeg
    PNG: .png
    GIF: .gif
    BMP: .bmp
    TIFF: .tif, .tiff
    ICO: .ico
    WebP: .webp
